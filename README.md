## Run Locally

Clone the project

```bash
  git clone https://github.com/ozgeahras/role-playing-game.git
```

Go to the project directory

```bash
  cd ..role-playing-game
```

Install dependencies

```bash
npm install
```

Start the app

```bash
npm run dev
```
